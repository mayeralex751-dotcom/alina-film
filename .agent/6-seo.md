# agent/6-seo.md — SEO Checklist "ReneeInspiredFilm"

On-page SEO:
- Unique title & meta description per page
- H1 = page title
- Alt text on all images
- Open Graph & Twitter Cards
- Canonical tags
- Structured data (Person, Photograph, Article)

Technical SEO:
- Sitemap.xml (auto-generated by Astro)
- robots.txt
- Fast loading (image optimization, lazy load)
- Mobile-first responsive
- HTTPS (free on Netlify/Vercel)

Content SEO:
- Target keywords: "film photographer [city]", "analog photography [style]", "35mm film portraits"
- Blog posts 800–1500 words
- Internal linking
- Image file names descriptive (e.g. film-portrait-bangkok.jpg)

Tools to use:
- Astro SEO components
- Schema.org JSON-LD
- Google Search Console (later)
