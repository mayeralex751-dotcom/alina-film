# agent/4-tech-stack.md — Technology Stack "ReneeInspiredFilm"

Core Framework
- Astro 5.x (latest stable) – excellent SEO, fast, component islands

Styling
- Tailwind CSS 3.x – rapid design, utility-first

Form & CRM
- Formspree / Getform / Netlify Forms – email notifications
- JSON file or Airtable (free tier) – lead storage

Deployment
- Netlify / Vercel / Cloudflare Pages – free, fast, auto-SSL

Development Tools
- VS Code + Astro extension
- Prettier + ESLint (optional)

Languages
- HTML + Astro components
- Tailwind for CSS
- JavaScript only when needed

Do not use:
- React / Vue / Svelte full (only islands if needed)
- Heavy CMS (WordPress, etc.)
- Server-side runtime (Node.js server)

Preferred form service: Formspree (free tier 50 submissions/month)
